This document is a brief overview of the CS185 Spring Quarter Project Scope. More details on requirements will be announced shortly.

  1. Canned and Custom D3JS web-based visualizations of of prescribed data.
  2. Use of the Stardust GPU visualization library
  3. Design of formative lab-based user study to evaluate data visualization and interaction (pre, main and post evals)
  4. Use of online questionnaires such as Qualtrics to elicit feedback from users.
  5. Implementation of the user study (in-class event)
  6. Implementation of MTurk Crowdsourced study (in-class event, selected projects only)
  7. Statistical Analysis and write-up of results (group homework)

Objective:

Design and implementation of a real-world application, with special focus on UI design.
The UI to your application is the important part of this project. It should apply suitable design guidelines and principles you learned in class.
